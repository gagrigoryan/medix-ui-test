### Использование:
Для использования следует импортировать компонент конкретной иконки из `marketplace-ui-kit`.

<Source
    language='tsx'
    dark
    format={true}
    code={`
        import {StarIcon, EIconSize} from "marketplace-ui-kit";
        ...
        <StarIcon size={EIconSize.XSmall} />
        <StarIcon size={EIconSize.Small} />
        <StarIcon /> // <StarIcon size={EIconSize.Normal} />
        <StarIcon size={EIconSize.Large} />
  `}
/>
