### Использование:
Для использования следует импортировать компонент `Notification` из `marketplace-ui-kit`.

<Source
    language='tsx'
    dark
    format={true}
    code={`
        import {Notification} from "marketplace-ui-kit";
        ...
        <Notification text="Sample notification text" />
        <Notification text="Sample notification text" onClose={() => {}} /> // Closable
        <Notification
            text="Sample notification text"
            actionText: "Cancel"
            onActionClick={() => {}}
            onClose={() => {}}
        /> // With action
  `}
/>
