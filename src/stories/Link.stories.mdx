### Использование:

Для использования следует импортировать модуль стилей `link.module.scss` из `marketplace-ui-kit/lib/styles`.
<Source
    language='css'
    dark
    format={true}
    code={`
        @import "~marketplace-ui-kit/lib/styles/link.module.scss";
        .myLink {
            composes: mxLink;
        }
        .myLinkLight {
            composes: mxLinkLight;
        }
  `}
/>
