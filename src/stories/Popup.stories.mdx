### Использование:
Для использования следует импортировать компонент `Popup` из `marketplace-ui-kit`.

<Source
    language='tsx'
    dark
    format={true}
    code={`
        import {Popup, EPopupSize} from "marketplace-ui-kit";
        ...
        <Popup
            title="Header"
            buttonLabel="Save"
            secondaryButtonLabel="Cancel"
            size={EPopupSize.large}
            isClosable
            footerContent="Footer Content"
        >
            контент попапа
        </Popup>
  `}
/>

Данный компонент принимает общий вид, если Вам нужно поменять отображение шапки и подвала попапа, то стоит использовать компонент [PopupBase](?path=/docs/popup-popupbase--base) в связке с `PopupLayout`.
<Source
    language='tsx'
    dark
    format={true}
    code={`
        import {PopupLayout, PopupBase} from "marketplace-ui-kit";
        ...
        <PopupLayout>
             <PopupBase {...props} />
        </PopupLayout>
  `}
/>
